## 1. 웹사이트 시작하기

웹사이트는 컴퓨터에서 실행됩니다. 매번 컴퓨터를 켤 때마다 아래 절차를 따라야 해요.

### 순서

1. **검은 창(터미널/PowerShell)을 엽니다**
   - 윈도우 검색창에서 `PowerShell` 검색 → 클릭

2. **아래 명령어를 입력하고 Enter**
   ```
   cd C:\Users\user\Desktop\lumiere-app
   npm run dev
   ```

3. **"ready" 또는 "Local: http://localhost:3000" 메시지가 나오면 성공**

4. **인터넷 브라우저(Chrome 등)에서 아래 주소로 접속**
   ```
   http://localhost:3000
   ```

> 이 창을 닫으면 웹사이트가 꺼집니다. 웹사이트를 켜놓을 때는 이 창을 최소화만 하세요.

---

## 2. 관리자 페이지 접속

관리자 페이지에서 제품을 추가·수정·삭제할 수 있어요.

1. 브라우저에서 이 주소로 이동:
   ```
   http://localhost:3000/en/admin
   ```

2. 비밀번호 입력창이 나타납니다. 비밀번호를 입력하고 로그인.
   - 기본 비밀번호: `changeme123`
   - (비밀번호는 `.env.local` 파일의 `ADMIN_PASSWORD` 값입니다)

3. 로그인 성공 시 제품 목록 테이블이 나타납니다.

---

## 3. 제품 수정하기

### 가격 바꾸기

1. 관리자 페이지에서 바꾸고 싶은 제품을 찾습니다
2. 해당 제품 줄에서 **Edit(수정)** 버튼을 클릭
3. 가격(Price) 칸을 클릭해서 새 숫자를 입력
4. **Save(저장)** 버튼 클릭 → 즉시 반영됩니다

### 제품 이름 바꾸기

1. **Edit** 버튼 클릭
2. Name(이름) 칸 수정
3. **Save** 클릭

### 재고 상태 바꾸기

1. **Edit** 버튼 클릭
2. **In Stock** 체크박스를 클릭해서 켜고 끄기
   - ✅ 체크 = 재고 있음 (카탈로그에 표시)
   - ☐ 체크 해제 = 품절 (품절 표시로 변경)
3. **Save** 클릭

### 제품 설명·스펙 바꾸기

1. **Edit** 버튼 클릭
2. Spec(스펙), Description(설명) 칸 수정
3. **Save** 클릭

## 4. 새 제품 추가하기

현재 관리자 페이지는 기존 제품 수정만 지원합니다.
새 제품은 아래 방법으로 추가하세요.

### 방법: `data/products.json` 파일 직접 수정

1. 탐색기에서 `C:\Users\user\Desktop\lumiere-app\data\products.json` 파일을 엽니다
   - 메모장(Notepad)이나 VS Code로 열기

2. 파일 끝부분에서 마지막 제품 항목을 찾아 그 뒤에 복사해서 붙여넣기:
   ```json
   {
     "id": 439,
     "name": "새 제품 이름",
     "category": "카테고리 이름",
     "price": 99.99,
     "spec": "용량/규격",
     "description": "제품 설명",
     "image": "",
     "tags": [],
     "inStock": true
   }
   ```
   > `id`는 기존 최대값 + 1로 설정 (예: 438개면 439)

3. 저장 후 웹사이트 새로고침

## 5. 제품 삭제하기

1. 관리자 페이지에서 삭제할 제품을 찾습니다
2. 해당 줄의 **Delete(삭제)** 버튼 클릭
3. 확인 팝업이 나오면 **OK** 클릭
4. 즉시 삭제됩니다

> **주의**: 삭제한 제품은 복구할 수 없어요. 신중하게 삭제하세요.

## 6. 연락처·소셜·결제 정보 바꾸기

이메일, WhatsApp 번호, Telegram 아이디, USDT 지갑 주소 등은
`lib/site-config.ts` 파일 하나에 모두 모아두었습니다.

### 수정 방법

1. `C:\Users\user\Desktop\lumiere-app\lib\site-config.ts` 파일을 엽니다
2. 변경하고 싶은 항목을 찾아서 값을 바꿉니다:

```ts
contact: {
  email: "info@lumiere-aesthetic.com",      // ← 이메일 주소
  whatsapp: "+82-10-0000-0000",             // ← WhatsApp 번호
  telegram: "@lumiere_aesthetic",            // ← Telegram 아이디
  address: "Seoul, South Korea",            // ← 회사 주소
},

social: {
  whatsapp: "https://wa.me/821000000000",   // ← WhatsApp 링크 (숫자만, + 없이)
  telegram: "https://t.me/lumiere_aesthetic",
},

payment: {
  usdt: {
    address: "TRC20_ADDRESS_PLACEHOLDER",   // ← USDT 지갑 주소 (TRC-20)
  },
},

shipping: {
  fedexWithAccount: 35,                     // ← FedEx 계정 있을 때 요금($)
  fedexWithoutAccount: 65,                  // ← FedEx 계정 없을 때 요금($)
},
```

3. 파일 저장 → 웹사이트 자동으로 반영됩니다 (개발 서버 실행 중일 때)

### WhatsApp 링크 만드는 법
- 번호: `+82-10-1234-5678` → 링크: `https://wa.me/821012345678`
- (국가코드 포함, + 와 - 기호 제거)

## 7. 주문 처리 방법

고객이 웹사이트에서 주문을 완료하면:

1. **고객이 결제 페이지에서 주문번호(예: `LUM-20260510-4821`)를 받습니다**
2. **고객이 직접 결제를 진행합니다:**
   - **Wise**: 계좌 정보로 은행 이체
   - **USDT**: 표시된 TRC-20 지갑 주소로 송금
3. **고객이 WhatsApp 또는 Telegram으로 연락합니다:**
   - 주문번호
   - 결제 완료 스크린샷 또는 해시
   - 배송 주소
4. **확인 후 배송 처리**

> 웹사이트는 결제를 자동으로 확인하지 않아요. 모든 결제 확인은 직접 하셔야 합니다.

## 8. 자주 묻는 질문 (FAQ)

### Q. 웹사이트를 껐다가 켜면 제품 정보가 사라지나요?
아니요. 모든 제품 정보는 `data/products.json` 파일에 저장됩니다. 컴퓨터를 꺼도 유지됩니다.

### Q. 제품 이미지를 추가하려면 어떻게 하나요?
1. 이미지 파일(JPG/PNG)을 `public/images/products/` 폴더에 복사
2. 관리자 페이지에서 해당 제품 Edit → Image URL 칸에 `/images/products/파일명.jpg` 입력
3. Save 클릭

### Q. 언어를 바꾸려면 어떻게 하나요?
웹사이트 오른쪽 상단의 언어 선택 드롭다운(EN/RU/KO)을 클릭하세요.

### Q. 한국에서는 접속이 안 되나요?
네, 한국 IP에서 결제 페이지 접근 시 제한 안내가 표시됩니다. (수출 전용 사이트)

### Q. 비밀번호를 바꾸고 싶어요
`.env.local` 파일을 열어서 `ADMIN_PASSWORD=새비밀번호` 로 바꾸고 저장 후 서버를 재시작하세요.

### Q. 웹사이트가 갑자기 안 켜져요
터미널에서 아래 명령어를 순서대로 실행해보세요:
```
cd C:\Users\user\Desktop\lumiere-app
npm install
npm run dev
```

### Q. 카탈로그 PDF 파일을 교체하고 싶어요
새 PDF를 `public\APR2026-CATALOGUE.pdf` 이름으로 저장하면 됩니다 (기존 파일 덮어쓰기).
